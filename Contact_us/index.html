Frequently Asked Questions

Are the calculators free to use?▼
Yes — all calculators are completely free. No login, no subscription and no paywall. The website is supported by advertisements.
Is this a certified design software?▼
No. Construction-Calculator is a proof-checking and verification tool — not a certified design platform. All results must be reviewed and stamped by a licensed structural engineer before use in actual construction. The tool is intended to help engineers quickly verify calculations and check IS code compliance during the design review process.
Which IS codes are referenced?▼
RCC design calculators reference IS 456 : 2000 and IS 13920 : 2018. Seismic calculators reference IS 1893 : 2016 and IS 1893 : 2025. All clause numbers are shown alongside results so you can verify directly against the code.
I found a calculation error — what should I do?▼
Please email us at construction.calculators@gmail.com with subject "Formula / Calculation Error". Include your input values, the result you received and the correct result you expect. We take accuracy very seriously and will investigate promptly.
Can I request a specific calculator?▼
Yes. Email us at construction.calculators@gmail.com with subject "Calculator Request" and mention the IS code reference and type of calculator needed. We prioritise based on the number of requests received.
Is there an Android app?▼
Yes. The RCC Design Calculator (IS456) app is available on Google Play Store. It currently includes the Singly Reinforced Beam Design module with live cross-section sketch and 12 IS code checks. More modules are being added.
